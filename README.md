# Meus Sites Favoritos

Este é um projeto que fiz para a faculdade de Análise e Desenvolvimento de Sistemas. A ideia foi criar uma página simples para guardar os links dos sites que eu mais uso para estudar e me divertir.

# O que usei:
* Estrutura em HTML para montar a página.
* Estilo CSS para organizar as imagens e o visual.

# O que tem na pasta:
* O arquivo do site (`index.html`).
* As imagens dos logos que baixei (YouTube, GitHub, Canva, Python e Stack Overflow).
* Fotos (prints) mostrando o código que escrevi e como o site ficou no navegador.
